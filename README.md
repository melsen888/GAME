# GAME
A simple javascript game

### Installation
1. Clone the repo 
```sh 
git clone https://github.com/melsen888/GAME
```
2. Install NPM packages
```sh
npm install
```

## Usage

> ```sh
> npm start
> ```
 or 
> ``` sh 
> node app.js
> ```
